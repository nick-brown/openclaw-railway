# Heartbeat Checklist

Run through this checklist each heartbeat cycle. Reply HEARTBEAT_OK if
nothing needs attention. Otherwise, surface only what's actionable.

## Email

- Scan Gmail inbox for new messages since last check.
- Flag anything urgent or time-sensitive.
- Summarize new messages worth noting (sender, subject, one-line summary).
- If an email contains an actionable task, create it on the Notion Personal Board.

## Calendar

- Check today's and tomorrow's events for conflicts or upcoming deadlines.
- Surface any events in the next 2 hours that need preparation.
- Flag any calendar invites that need a response.

## Notion Tasks

- Check the Personal Board (YOUR_NOTION_BOARD_ID) for:
  - Tasks due today or overdue
  - Tasks with upcoming deadlines (next 3 days)
- Update task status if you have new information (e.g., an email confirms
  something is done).

## Rules

- Don't repeat information already surfaced in a previous heartbeat.
- Keep summaries to one line per item.
- If nothing needs attention across all checks, reply HEARTBEAT_OK.
